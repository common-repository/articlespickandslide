Requires at least: 3.3.1
Tested up to: 3.3.1

== Description ==

Adds an article slider on the sidebar of your website which shows articles from a specific category

== Installation ==


Installation of this plugin is fairly easy:

1. Extract all the files. 
2. Upload everything (keeping the directory structure) to the `/wp-content/plugins/` directory.
3. There should be a `/wp-content/plugins/articlesPickAndSlide` directory now with `articlesPickAndSlide.php` in it.
4. Activate the widget through the 'Plugins' menu in WordPress.
5. Go to Appearance/Widgets and drag and drop the articlesPickAndSLide widget in a bar
6. Set the title of the Widget
7. Choose the name of the Category from the articles should be picked
8. Set the maximum number of articles to be shown

The articles are publish in descendent order by the publishing date.
